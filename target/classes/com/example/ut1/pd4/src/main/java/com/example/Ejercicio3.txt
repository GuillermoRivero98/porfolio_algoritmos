El siguiente código crea un array y una string. ¿Cuántas referencias a estos objetos
existen luego de que el código se ha ejecutado? ¿Es alguno de los objetos candidato a
ser eliminado por el garbage collector?

 String[] students = new String[10];
 String studentName = "Peter Parker";
 students[0] = studentName;
 studentName = null;
 ...

Existen 2 referencias a objetos 
    students, una referencia al array de strings.
    students[0], que apunta a "Peter Parker".
    studentName es null, pero "Peter Parker" sigue referenciado por students[0].

Candidato para Garbage Collector:
    Ninguno, ya que "Peter Parker" aún es referenciado por students[0].
